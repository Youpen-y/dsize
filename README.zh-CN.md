# dsize

**[English](README.md) | [中文](README.zh-CN.md)**

`dsize` 是一个简易的 Bash 脚本工具，用于以美观、易读的格式显示文件和目录的大小。相比于传统的 `du` 命令，它提供了更好的视觉反馈、颜色区分以及对中文字符对齐的优化支持。

![演示截图](./screenshot.png)

---

## ✨ 核心特性

* **人类可读的格式**：自动将字节转换为 B、K、M、G、T 等单位，并根据大小级别显示不同颜色（如绿色表示 KB，黄色表示 MB，红色表示 GB/TB）。
* **智能对齐**：针对终端显示进行了优化。如果系统中安装了 Python 3，脚本将精确计算中文字符宽度，确保文件名与大小数据严格对齐。
* **深层查看**：支持展示目录下的子项大小（嵌套深度为 1），方便快速定位空间占用。
* **排序功能**：支持按文件大小进行降序排列（`--sort`），让"空间杀手"一目了然。
* **符号链接支持**：自动解析软链接，显示链接指向的真实路径。
* **色彩开关**：支持 `-c` 选项禁用颜色输出，方便在自动化脚本或导出日志时使用。

## 🚀 安装指南

1. **保存脚本**：下载 `dsize` 文件。
2. **赋予执行权限**：
    ```bash
    chmod +x dsize
    ```
3. **移动到系统路径（可选）**：
    为了方便在任何目录下使用，建议将其移动到 `/usr/local/bin`：
    ```bash
    sudo mv dsize /usr/local/bin/
    ```

## 🆚 为什么选择 dsize 而不是 du？

`dsize` 本质上是 `du` 命令的增强包装器，解决了常见的痛点：

| 功能 | `du` 命令 | `dsize` |
|---------|--------------|---------|
| **人类可读输出** | 需要 `-h` 参数，且会破坏排序 | 始终人类可读，排序正确 |
| **颜色编码** | 无 | 基于大小的颜色（绿/黄/红） |
| **CJK 字符对齐** | 文件名错位 | 智能宽度计算 |
| **符号链接** | 显示链接本身 | 解析并显示真实路径 |
| **排序** | 需要管道到 sort | 内置 `-s` 参数 |
| **视觉美观** | 原始数字 | 带边框的美观输出 |
| **内存效率** | — | 大目录流式处理 |

### 颜色编码指南

| 大小范围 | 颜色 | 含义 |
|------------|-------|---------|
| B (字节) | 白色 | 小文件 |
| KB | 绿色 | 小文件 |
| MB | 黄色 | 中等文件 |
| GB/TB | 红色 | 大文件 - 需要关注 |

## 📖 使用方法

### 命令格式
```bash
dsize [选项] <路径>
```

### 常用选项
| 选项 | 长选项 | 描述 |
|---|---|---|
| `-d` | `--detail` | 显示子目录/文件大小（不含隐藏文件） |
| `-v` | `--verbose` | 显示所有子目录/文件（包含隐藏文件） |
| `-s` | `--sort` | 按大小降序排列（需配合 `-d` 或 `-v` 使用） |
| `-c` | `--no-color` | 禁用彩色输出 |
| `-h` | `--help` | 显示帮助信息 |

### 示例演示

- 查看当前目录总大小：
```bash
dsize .
```

- 查看子目录详情并排序：
```bash
dsize -d -s /var/log
```

- 包含隐藏文件查看用户家目录：
```bash
dsize -v ~
```

- 查看单个文件信息：
```bash
dsize file.txt
```

## 🛠 依赖要求

- **Bash**：脚本运行的核心环境。
- **awk**：用于精确的浮点数计算。
- **Python 3**（可选）：用于处理 CJK（中日韩）字符的对齐。如果未安装，脚本将退回到基于字节数的估算对齐。
